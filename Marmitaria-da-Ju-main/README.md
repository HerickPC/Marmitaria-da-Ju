# Projeto-Site-Ju-Marmitaria

Este repositório foi desenvolvido como parte das atividades acadêmicas da FATEC São Roque, com o objetivo de aplicar na prática os conhecimentos adquiridos em sala de aula por meio do desenvolvimento de um website institucional para a Ju Marmitaria.

O projeto busca integrar conceitos relacionados ao desenvolvimento de sistemas web, organização de dados, design de interfaces e boas práticas de programação, proporcionando experiência prática na criação e estruturação de aplicações voltadas ao ambiente digital.

Além do desenvolvimento técnico, o projeto também visa estimular habilidades como trabalho em equipe, versionamento de código utilizando Git e GitHub, planejamento de interface e organização de projetos.
